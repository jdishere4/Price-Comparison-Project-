## Project Overview
The Price Comparison Tool is a Python-based web scraping project that extracts laptop 
product data from multiple e-commerce websites and compares prices.
The project scrapes product Name, Price, and Ratings for a selected laptop model
and generates a comparison report along with a visualization of price differences.

## Objective
To demonstrate practical skills in:
- Web Scraping
- Data Cleaning
- Data Comparison
- Data Visualization
- Exporting structured reports
## Technologies Used
- Python
- BeautifulSoup / Selenium
- Pandas
- Matplotlib
